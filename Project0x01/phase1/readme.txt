1. 컴파일 방법
4 파일이 모두 포함된 디렉토리에서 make command를 입력한다.
myshell.c

2. 실행 방법
컴파일 완료 후 ./myshell을 입력하여 프로그램을 실행시킨다.
프로그램이 실행되면 CSE4100-MP-P1>이 출력된다.
원하는 명령어를 입력하여 사용한다.

command example)
cd : 디렉토리 탐색 및 이동
